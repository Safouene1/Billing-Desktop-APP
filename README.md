# Billing System 

a simple Web / Desktop App for a dentical Lab. 


## Build / Run 

To run / build  this project run

```bash
  npm run serve
  npm run build
```

  
## Tech Stack

Made with Vue 3 (Optional API) , Vue Router 4 , Vuex 4 , Boostrap 5 , Electron 12
